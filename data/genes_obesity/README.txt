Added gene lists from Locke2015 Table S4
Gene lists generated by uploading n=97 GWAS SNPs to SNPsnap (https://data.broadinstitute.org/mpg/snpsnap) with default loci definition. 
n=95 SNPs were found in SNPsnap.
For "nearest_genes" we used the SNPsnap column ID_nearest_gene_snpsnap_protein_coding (we are not interested in non-coding genes)
For "associated_genes" we used the SNPsnap column ID_genes_in_matched_locus


locke2015_table_s4.gwas_loci_n97.snpsnap.txt: SNPsnap selected output

locke2015_table_s4.gwas_loci_n97.nearest_genes.txt: nearest protein coding gene

locke2015_table_s4.gwas_loci_n97.associated_genes.txt: all genes in associated loci
